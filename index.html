Cómo usar datos completos

La web puede funcionar con el CSV de estados fundamentales de IAEA LiveChart. Pulsa Cargar datos IAEA. Si el navegador bloquea la petición por CORS, descarga el CSV desde la API de IAEA y usa Importar CSV.

https://nds.iaea.org/relnsd/v1/data?fields=ground_states&nuclides=all

Campos esperados

El normalizador acepta columnas habituales como z, n, symbol, element, half_life_sec, half_life, unit_hl, decay_1, decay_1_%, atomic_mass, mass_excess, abundance, jp, energy y conserva cualquier columna extra en el panel de detalle.
